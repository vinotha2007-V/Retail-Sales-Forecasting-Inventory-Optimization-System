# 🛒 Retail Sales Forecasting & Inventory Optimization System

## 📌 Overview
This project predicts retail sales and optimizes inventory levels using machine learning.

## 🚀 Features
- Sales Forecasting using ML
- Inventory Optimization
- Demand Prediction
- Flask Web App

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Flask

## 📊 Workflow
1. Data Preprocessing
2. Feature Engineering
3. Model Training
4. Forecasting
5. Inventory Optimization

## ▶️ How to Run

```bash
pip install -r requirements.txt
python src/train_model.py
python src/forecast.py
python src/inventory_optimizer.py
python app/app.py
